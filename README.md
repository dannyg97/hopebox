# HopeBox
A mood tracker app that shows healthy alternatives to negative emotions. 

### What is HopeBox? 
HopeBox is an iOS app that focuses on the tracking and improvement of mental health. 


### Technology Sets 
IDE: Xcode 
Language: 
- Swift for frontend 
- Python for backend
